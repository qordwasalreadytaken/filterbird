FilterPhoenix Filter Builder

Visual Loot Filter Builder for Path of Diablo

What is FilterPhoenix Filter Builder?

FilterPhoenix Filter Builder is a web-based visual editor for creating custom loot filters for Path of Diablo. It provides an intuitive interface to customize which items you see in-game, with support for colors, notifications, filter levels, and advanced filtering options.

  • Set Items: High-level, mid-level, and low-level set pieces

  • Rare Items: Jewelry, class-specific items, and elite bases
  • NMAG (Non-Magic) Items: Runeword bases for all weapon and armor types

  • Gems & Misc: Gems, skulls, maps, and PoD items

üé® Customization Options

  • Custom Colors: Set font color, border color, and background color for any item
  • Chat Notifications: Enable in-game chat notifications when specific items drop
  • Bigger Clickboxes: Make items easier to click in-game with expanded display areas
  • NMAG Filters: Filter by ethereal status, socket count (0-6), and enhanced damage ranges

How to Use

Basic Workflow

  1. Enable/Disable Items: Check or uncheck items to include them in your filter
  2. Set Filter Levels: If using filter levels, check which levels each item should appear in
  3. Customize Appearance: Click the ‚öô gear icon to open advanced settings for colors, notifications, and display options
  4. Use Presets: Apply quick presets for common use cases (e.g., "Show High Runes Only", "All Gems")
  5. Copy Filter: Click "Copy Filter to Clipboard" to get your complete filter code
  6. Save to File: The filter automatically saves to FPCustom.filter in your downloads

üí° Filter Levels

Filter levels allow you to create multiple "views" of your loot in a single filter. Enable filter levels, name them (e.g., "1-70 Leveling", "80+ Endgame"), and then assign items to specific levels. Use the All/None buttons for quick selection.

When filter levels are disabled, all items are shown by default (same as selecting all levels).

Installation

  1. Generate your filter using FilterPhoenix Filter Builder
  2. Copy or download the FPCustom.filter file
  3. Place it in your Filter folder in the Path of Diablo game folder (where PoD Launcher is located)
  4. Launch the game and pick your filter from the H menu

‚ö†Ô∏è Important Notes

  • Backup your existing filter before replacing it
  • Test your filter in-game to ensure it works as expected
  • The filter respects Path of Diablo's filter syntax - invalid syntax may cause the filter to fail silently

Tips & Tricks

  • Start Simple: Begin with presets and gradually customize to your needs
  • Use Filter Levels Wisely: Create levels for different characters or playstyles
  • Test Incrementally: Make changes, test in-game, then iterate
  • Color Coding: Use consistent color schemes (e.g., orange for valuable, white for bases)
  • NMAG Filters: Be specific with socket counts for runeword bases (e.g., 4 sockets for Spirit)
